# M7：2D DDPM 生成 ct28 孔隙结构（MVP 首基线）

> 阶段二第一个生成实验。依据见 [[20_阶段二_2D扩散启动设计]]（M5★三段依据 + M6）、精读卡①②（[[02_文献综述]]）。
> 原则 [[provenance-no-fabrication]]：如实记录——**流水线跑通，但首基线质量不达标**。

## 0. 一句话结论 `[测]`
**端到端流水线（数据→DDPM→采样→参数评估）已验证可用；但首个 DDPM 基线过度生成孔隙约 3.7×（gen 23.5% vs real 6.4%），呈椒盐噪声状，结构不达标 → 需迭代（首疑：二值化阈值 + 欠训练）。**

## 1. 配置 `[测]`
- 数据：M7.1 切片器从 E2/E3 验证的同网格 ct28(2.8μm) 体出 2D 二值孔隙切片；128²，16600 训练 / 4150 测试（沿 z 顶部 20% 防泄漏留出）。孔隙度 train 7.29% / test **中位 5.67%、均值 6.40%**（分布右偏；§2/评估对真实集统一用**均值 6.40%**）。
- 模型：diffusers `UNet2DModel`（63.1M，base64，attn@16²）+ `DDPMScheduler`（=Ho2020：T=1000、β 线性 1e-4→0.02、ε-预测 L_simple、数据[-1,1]）。
- 训练：50 epoch、bs64、lr1e-4、AMP，5090；L_simple 收敛 0.019。
- 采样：1000 步反向，512 张；二值化 = 连续输出阈 0。

## 2. 参数评估（生成 512 vs 真实测试 512）`[测]`
| 指标 | 真实 | 生成 | 判读 |
|---|---|---|---|
| 孔隙度 % 均值 | 6.40 | **23.51** | KS p=0、Wasserstein 17.1 → 严重偏高 ~3.7× |
| S₂(0) (=φ) | 0.064 | 0.235 | 与孔隙度一致 |
| S₂(r) 形态 | 快速降到 φ²≈0.004 | 高位、降到 ~0.065 平台 | 孔隙密度整体过高 |
| Euler χ 均值 | 127.33 | 146.24 | 碎块更多（散点） |
- 视觉：真实=稀疏离散成簇的孔斑（~5-6%）；生成=**密集椒盐细斑**（~23%）。图：`experiments/花页7_PlanB_记录/phase2/{samples_grid,real_grid,fig_eval}.png`。

## 3. 诊断与迭代方向 `[假]`
"椒盐噪声 + 过度生成"强烈指向：
1. **二值化阈值**（首疑）：DDPM 连续输出在矩阵相(-1)附近有噪声，阈值取 0 时尾巴越过 0 → 散点假孔。**验证法**：重采样**保存连续值** → 把阈值**标定到复现真实孔隙度**，再看 S₂(r)/Euler 是否对齐——可分离"二值化伪影(可修)"与"结构真错(需训练)"。
2. **欠训练/表征**：50ep；可加 epoch，或改**灰度 sus 建模**（信息更丰，少二值化伪影），或用更适配稀疏二值的损失/表征。

> MVP 目的（验证生成机器 + 评估流水）已达成；质量迭代是下一个小目标 **M7-v2**，按纪律先做阈值标定诊断（便宜、决定性），再决定是否加训/换表征。
