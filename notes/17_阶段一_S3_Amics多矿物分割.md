# S3：Amics 多矿物 2D 分割（nnU-Netv2）

> 目的：在花页7单图像配对数据上补充多矿物定量内部评估（论文只在自建 SEM 上做，外部井只到 2–3 类）。
> 输入 = Amics «处理图像» 的 BSE 灰度；标签 = 同尺寸 Mineral_Image（QEMSCAN/EDS 矿物图）→ 论文口径 5 类。
> 颜色→矿物→5类映射经 `[核]` 面积比对粗扫矿物 vol% 确认，残差<0.05%，见 `experiments/花页7_PlanB_记录/evidence_amics_color_mapping.json`。
> 类定义（用户确认 2026-06-24）：0 背景 / 1 长英质 / 2 碳酸盐 / 3 粘土 / 4 金属重矿。
> 原则见 [[provenance-no-fabrication]]；数据真相见 memory `hy7-multiscale-label-inventory`。

## 0. 一句话结论 `[测]`
**同图像、无缓冲 Ts 下观察到尺度相关结果。** 25μm 与 1μm 的差异不能据此排除模型、FOV、tile 数量等因素；1μm 下丰富相（长英质/碳酸盐，占前景 92%）表现较高，稀有相仍受不平衡限制。

## 2. 科学判读（含诚实局限）

- 碳酸盐在 25μm 为 0.354/0.419、在 1μm 为 0.715/0.674；结果与 25μm 单通道 BSE 判别信息不足的解释一致，但不能单独证明原因或排除模型因素。
- 粘土/金属重矿即便 1μm 仍为 0.11–0.17；结果与类不平衡和单一小 FOV 数据量的解释一致，但不应写为非模型缺陷。
- **任务本质提醒** `[假]`：Mineral_Image 的矿物身份来自 **EDS 能谱(化学)**，BSE 灰度只是 Z 衬度的一个通道。"BSE→矿物"在硅酸盐内部（长英质 vs 粘土，皆低 Z）本就接近欠定 → 这是 1μm 下粘土仍难的深层原因之一，也指向"多通道/多模态"才是上限突破口（接阶段三）。

## 3. 与论文 T0613 的关系（中期汇报立论点）
- 论文：自建 SEM 0.3μm 六类 mDice 0.746；**外部井只能 2–3 类、CT 跨尺度只能定性**（无像素级真值）。
