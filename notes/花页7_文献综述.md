## B. 生成式三维重建（GAN，奠基）

- **Mosser, Dubrule, Blunt 2017 — Reconstruction of 3D porous media using GANs**
  arXiv [1704.03225](https://arxiv.org/abs/1704.03225)（PRE 96:043309）⊙ `literature/Mosser2017_3D_porous_media_GAN.pdf`
  - 要点：用 GAN 无监督学三维孔隙结构的概率分布，生成统计上代表性的多孔介质样本；附 GitHub 实现。
  - **对花页7**：E4 生成式起步的奠基方法——从分割得到的三维相场学分布、生成新样本。

- **Kench & Cooper 2021 — SliceGAN：从单张 2D 切片生成 3D 结构**
  arXiv [2102.07708](https://arxiv.org/abs/2102.07708)（Nature Machine Intelligence）⊙ `literature/Kench2021_SliceGAN_2D_to_3D.pdf`
  - 要点：仅用一张代表性 2D 图即可合成高保真 3D 体（x/y/z 切片喂 2D 判别器）；秒级生成 10⁸ 体素。
  - **对花页7 极相关**：花页7 有 2D 的 **Maps SEM(10nm)** 与纳米尺度切片——SliceGAN 思路可把
    **高分辨 2D → 3D**，正是"从少量扫描造连续数字岩心"主线的关键一招。

## C. 超分辨（细化分辨率）

- **2024 — Enhancing digital core image resolution（paired SEM）**
  arXiv [2409.03265](https://arxiv.org/abs/2409.03265) ⊙ `literature/2024_digital_core_SR_paired_SEM.pdf`
  - 要点：配对 SEM 图像的最优上采样/超分。
  - **对花页7**：E4 候选——把 2.8μm CT 超分到接近纳米尺度，连通多尺度孔隙度非单调的解释。
- 另检索到（未下，付费/综述，链接备查）：PPD-GAN、RDCA-SRGAN、残差 U-Net+多尺度融合 SR、
  "A Decade of GANs for Porous Material Reconstruction"(综述 arXiv 2603.11836)。

## D. 扩散模型（当前 SOTA 方向）

- **2025/2026 — Controlled Latent Diffusion Models for 3D Porous Media Reconstruction**
  arXiv [2503.24083](https://arxiv.org/abs/2503.24083)（Computers & Geosciences 206, 2026）⊙
  `literature/Liu2025_Controlled_Latent_Diffusion_3D_porous.pdf`
  - 要点：用 VAE 把二值地质体降维到隐空间 + EDM 框架潜扩散，能生成比以往扩散更大的体；可控生成。
  - **对花页7**：E4 的 SOTA 备选；其"VAE+潜扩散生成大体"正解花页7 体素大(1500³/1620³)的难处。
- 相关（未下）：Diffusion 生成 3D 多相孔隙图像(TiPM 2025)、有限岩心扫描+多模态融合的扩散 3D 重建(2025)。

## E. 多尺度对齐 / 配准（本工作的另一支柱）

- **2025 — Image registration of 2D optical thin sections in a 3D porous medium (Berea)**
  arXiv [2504.06604](https://arxiv.org/abs/2504.06604) ⊙ `literature/2025_2D_3D_registration_Berea_digital_rock.pdf`
  - 要点：把 2D 薄片配准进 3D 数字岩心体。
  - **对花页7**：E3「跨尺度对齐」的参考——Plan B 先用同网格绕开配准，跨尺度时仍需这类方法。
