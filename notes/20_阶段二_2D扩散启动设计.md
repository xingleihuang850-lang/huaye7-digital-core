## M5★ 三段依据（文献 / 物理 / 地质）—— 动手前的根据，禁无根据闷头做

> 按工作纪律（CLAUDE.md ★节）：设计任何步骤前先落实文献+物理+地质三段依据。本节为阶段二（2D 扩散）的依据。

**① 文献依据**
- **Ho2020（精读卡①）**：DDPM 的 ε-预测训练目标 `L_simple` + Algorithm1/2 采样，T=1000、β 线性 1e-4→0.02、数据 [-1,1] —— **MVP 生成器照此配**。
- **Mosser2017（精读卡②）**：多孔介质生成的**评估指标集** φ / S₂(r) / Minkowski(SV, χ) / LBM 渗透率 —— **MVP 评估照此（取 2D 版）**，确保"参数评估"而非只看图。
- SliceGAN / Naiff2025 潜扩散：2D→3D 与大体生成，阶段二→三的接续依据。

**② 物理意义**（为什么这些指标是对的，不是凑数）
- **孔隙度 φ**：储集能力（一阶量）。
- **两点相关 S₂(r)**：孔隙空间两点同处孔相的概率；r=0 给 φ，其衰减长度≈**相关长度/特征孔尺度**，刻画孔的大小与空间排布。
- **比表面 SV / 比周长**：孔–基质界面量，关联**吸附容量与渗流阻力**。
- **Euler 特征 χ（连通性）**：拓扑量，直接决定孔隙是否**连成可渗流通道**——页岩气可动性的物理前提。
- → 这四类分布都与真实不可区分 ⇒ 生成的孔隙结构在**储集-渗流物理**上可信，而非仅视觉相似。

**③ 地质意义**（Loucks2012 锚定生成对象）
- 花页7 = 4199.21m **钙质长英质页岩**。泥岩基质孔三类：**粒间孔(interP) / 粒内孔(intraP) / 有机质孔(OM)**，外加独立的**裂缝孔**；孔隙+裂缝 = 页岩气**渗流网络**；interP 与 OM 孔更易互连（∴ χ 连通性是关键判据，有地质根据）。
- **尺度自觉（承 S3 教训 [[17_阶段一_S3_Amics多矿物分割]]）**：2.8μm CT 只分辨 μm 级 → 生成对象 = **μm 级粒间孔 + 较大粒内孔 + 微裂缝（宏观流路骨架）**；纳米级 OM 孔在此尺度以下（需 65nm/FIB-SEM/10nm Maps，留阶段三）。MVP **诚实界定为"μm 尺度基质孔+微裂缝网络"，不冒充全孔径**。
- → 生成对象有明确**地质身份**（matrix pore network），评估时按孔型/连通对齐 Loucks，而非抽象纹理生成。

## M6 配对样本 / 数据管线设计

### M6.0 MVP 范围界定（先立住生成机器 + 评估流水，再加难度）
- **MVP = 单尺度无条件 2D DDPM**：在**已验证可信**的同网格 CT 分割体上取 2D 切片，学并生成 2D 孔隙结构。
  - 为什么先单尺度无条件：阶段一已证 ct28(2.8μm)同网格 sus/pore 干净可信（E2 0.939；E3 nnU-Netv2 2d 50ep Dice 0.9660，4例验证，本地证据见 `evidence_e3_nnunet_2d50ep_summary.json`），**数据不引入新风险**；先确认"扩散能学到花页孔隙分布 + 评估指标流水跑通",再上配对/条件(避免一上来配准+条件+生成三难叠加)。
- **逐级加难度**（用户既定偏好）：MVP → 条件(给孔隙度/相邻切片) → 跨尺度配对(micro-CT↔FIB-SEM,用 ⑤的配准) → 2D→3D(SliceGAN/潜扩散)。

### M6.1 数据源（MVP，全在 hy7-linux，无需新扫描）
| 来源 | 规格 | 切片产出 | 备注 |
|---|---|---|---|
| ct28 2.8μm 同网格体 | sus(灰度)+pore(二值) ~1500³ | 数千张轴向 2D 切片 | E2 已验证对齐,**首选** |
| ct14 14μm 同网格体 | sus+pore+feng(三相) | 同 | 含裂缝,可做"带裂缝孔隙结构"变体 |
| (后续)Maps SEM 10nm 2D | 无机孔/有机孔/无机缝/有机缝 | 2D 原生最细孔结构 | 条件/跨尺度阶段用 |

### M6.2 预处理流水（写成 `src/hy7_phase2_make_slices.py`，待 M7）
1. 读同网格体(复用 `hy7_planb_io.ScaleVolumes`) → 沿 z 取切片(可加 x/y 三向增样)。
2. 切 256² 瓦片；**按孔隙覆盖率过滤**(丢全基质瓦片,避免模型只学空白)。
3. 通道选择：(a) 二值 pore/matrix(先做,纯结构);(b) 灰度 sus 归一(后做,带纹理)。
4. **防泄漏空间留出**(沿用阶段一纪律):按体的某一空间分区留出测试块,固定 seed。
5. 产出 + sha256 + 切片统计(孔隙度直方图)存 `experiments/`。

### M6.3 模型与训练（MVP 配置草案，待 M7 落地）
- 框架：HuggingFace `diffusers` `UNet2DModel` + `DDPMScheduler`(Ho2020),T=1000，cosine 调度；或从零实现 DDPM(以 ①精读为准)。
- 分辨率 256²(或先 128² 快验)，AMP，5090 单卡；epoch/步数先小后大。
- 环境：复用 `nnunet_t28` 或新建 `diffusers` env(待定,M7 时确认 torch+diffusers 版本钉入 requirements)。

### M6.4 评估（**阶段二的硬通货 = 参数评估,非只看图**）
对生成 N 张 vs 真实切片，比较分布(KS/Wasserstein)：
- **孔隙度** %（一阶）；
- **两点相关 S₂(r)**（Mosser 核心,空间结构）；
- **孔径分布 PSD**、**比表面积**；
- **连通性 = Euler 数/欧拉特征**(孔隙是否贯通,呼应裂缝/渗流);
- 视觉对照网格图。
- 工具：`scikit-image`(已装)+ 自写形态学；通过=各统计量分布与真实不可区分(或差异有界并诚实报告)。
