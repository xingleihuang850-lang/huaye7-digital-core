# 花页7 中期汇报 · 工作梳理

> 供拼中期汇报 PPT。主线四阶段：**① 分割(地基) → ② 2D 扩散生成 → ③ 多模态 3D → ④ 数字井筒**。
> 当前位置：阶段一已收尾，阶段二(2D 扩散)已起步并出首基线。下面 ✅PPT = 建议进汇报。
> 数字均 `[测]` 实测、可复现（脚本+证据+sha256 见各实验笔记）；codex/官页15 项为既有材料，已注明出处。

## 0. 一句话主张（汇报主线）
> 开题分割停在 **2D、单井、矿物只能内部验证**（论文 T0613）。花页7 提供**配准 CT 体 + 3D Avizo 标签 + Amics 2D 矿物 GT**，使我们：① 获得同网格路线与较高内部 Dice 一致的受限证据(E2/E3)；② 获得同图像尺度相关的多矿物内部评估(S3)；③ **跨过"分割→生成"那一跳**，建立 2D 扩散生成+参数评估流水线(M5–M7) → 为阶段三/四准备受限候选底料。

## 3. 原理与方法 + 实现 + 结果（按实验）

### 3.1 阶段一 · CT 三相/二相分割 + 配准排错（E0/E1/E2/E3）`[测]`
- **原理**：CT 灰度≈密度×Z；孔隙/裂缝灰度≈0（低密度空腔），裂缝-孔隙靠**几何/连通性**而非灰度分（裂缝难根因）。分割=体素级判别（U-Net/nnU-Netv2 编码-解码+跳连）。
- **方法/实现**：自写 3D UNet（在线 patch 采样，AMP；有效区>50%、IGNORE 柱塞外处理）；nnU-Netv2（数据指纹自配置，CTNormalization）。E1 用 CE 类加权+裂缝过采样。E3 为同工具受限对照。历史 E0–E3 未保存空间切分证据，不能称防泄漏。
- **结果（受限对照）**：

  | 工具/配置 | 图像 | 配准 | 孔隙 Dice |
  |---|---|---|---|
  | codex nnU-Netv2 3d/2d | 2024³裁剪 | 错位 | **≈0**（0.0 / 0.0002） |
  | E2 自写UNet | 同网格 sus | 共配准 | **0.939**（mean 0.9675） |
  | E3 nnU-Netv2 2d/3d | 同网格 sus | 共配准 | **0.966 / 0.919** |

  → 在这些受限对照中，同网格处理与较高内部 Dice 一致；不能归因为唯一变量，也不代表空间或跨井泛化。E0 三相同体内部 mean 0.831；E1 显示本次暴力重平衡设置得不偿失，不能据此确定裂缝上限。
- **图**：`fig_E2_vs_codex_ct28.png`（✅PPT）、ct14 配准切片 `ct14_z*.png`。

### 3.2 阶段一 · Amics 多矿物 2D 分割（S3）`[测]`
- **原理**：矿物身份来自 BSE 的 Z 衬度（+QEMSCAN/EDS）。25μm 与 1μm 结果可用于观察尺度相关差异，不能据此单独确定原因。
- **方法/实现**：Amics BSE↔Mineral 同尺寸配对，颜色→矿物→论文5类映射（`[核]`面积比对粗扫 vol%，残差<0.05%）；nnU-Netv2 2D 50ep；同图像列向无缓冲 Ts，训练/测试 tile 不重叠但仍可能空间相关。
- **结果（25μm vs 1μm，同图像无缓冲 Ts）**：

  | 类(占前景) | 25μm | 1μm |
  |---|---|---|
  | 长英质(49–55%) | 0.51 | **0.73** |
  | 碳酸盐(41–43%) | 0.42 | **0.67** |
  | 粘土(3–5%) | 0.00 | 0.11 |
  | 金属重矿(1–3%) | 0.04 | 0.13 |
  | 前景均值 | 0.24 | **0.41** |

  → 同图像无缓冲 Ts 结果显示尺度相关差异；不能排除邻近空间相关性、模型或 FOV 因素，亦非独立井泛化。稀有类与不平衡有关的解释仍需更多证据。
- **图**：`fig_S3_25um_vs_1um.png`（✅PPT）。

### 3.3 阶段二 · 2D 扩散生成孔隙结构（M5–M7）`[测]`
- **原理（DDPM, Ho2020）**：前向固定加噪 `q(xₜ|x₀)=N(√ᾱₜx₀,(1-ᾱₜ)I)`；反向学去噪，**ε-预测** `L_simple=E‖ε−εθ(√ᾱₜx₀+√(1-ᾱₜ)ε,t)‖²`；T=1000、β线性1e-4→0.02、数据[-1,1]。
- **物理/地质依据**：评估用 φ/两点相关 S₂(r)/Euler χ（连通性=渗流通道前提）；生成对象按 Loucks2012 锚定为页岩**基质孔+微裂缝网络**；2.8μm 只分辨 μm 级孔（诚实界定，不冒充纳米有机孔）。
- **方法/实现**：同网格 ct28→128² 二值孔隙切片 16600/4150（z 顶部防泄漏留出）；diffusers `UNet2DModel`(63M)+`DDPMScheduler`(=Ho2020)，50ep；采样 512 张→按 Mosser 指标对真实测试集评估。
- **结果（首基线，如实）**：流水线端到端通 ✅；但**过度生成孔隙 ~3.7×**（gen 23.5% vs real 6.4%，KS p=0；S₂(0) 0.235 vs 0.064；Euler 146.24 vs 127.33），呈椒盐噪声。诊断：首疑二值化阈值(连续输出阈0→矩阵相噪声成散点假孔)+欠训练；迭代 M7-v2 先做阈值标定诊断。
- **图**：`phase2/fig_eval.png`、`samples_grid.png`、`real_grid.png`（△PPT：作"生成式起步/流水线已建、首基线诊断"呈现，非成品）。
