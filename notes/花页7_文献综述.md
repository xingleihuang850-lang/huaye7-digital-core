## B. 生成式三维重建（GAN，奠基）

- **Mosser, Dubrule, Blunt 2017 — Reconstruction of 3D porous media using GANs**
  arXiv [1704.03225](https://arxiv.org/abs/1704.03225)（PRE 96:043309）⊙ `literature/Mosser2017_3D_porous_media_GAN.pdf`
  - 要点：用 GAN 无监督学三维孔隙结构的概率分布，生成统计上代表性的多孔介质样本；附 GitHub 实现。
  - **对花页7**：E4 生成式起步的奠基方法——从分割得到的三维相场学分布、生成新样本。

- **Kench & Cooper 2021 — SliceGAN：从单张 2D 切片生成 3D 结构**
  arXiv [2102.07708](https://arxiv.org/abs/2102.07708)（Nature Machine Intelligence）⊙ `literature/Kench2021_SliceGAN_2D_to_3D.pdf`
  - 要点：仅用一张代表性 2D 图即可合成高保真 3D 体（x/y/z 切片喂 2D 判别器）；秒级生成 10⁸ 体素。
  - **对花页7 极相关**：花页7 有 2D 的 **Maps SEM(10nm)** 与纳米尺度切片——SliceGAN 思路可把
    **高分辨 2D → 3D**，正是"从少量扫描造连续数字岩心"主线的关键一招。

## D. 扩散模型（当前 SOTA 方向）

- **2025/2026 — Controlled Latent Diffusion Models for 3D Porous Media Reconstruction**
  arXiv [2503.24083](https://arxiv.org/abs/2503.24083)（Computers & Geosciences 206, 2026）⊙
  `literature/Liu2025_Controlled_Latent_Diffusion_3D_porous.pdf`
  - 要点：用 VAE 把二值地质体降维到隐空间 + EDM 框架潜扩散，能生成比以往扩散更大的体；可控生成。
  - **对花页7**：E4 的 SOTA 备选；其"VAE+潜扩散生成大体"正解花页7 体素大(1500³/1620³)的难处。
- 相关（未下）：Diffusion 生成 3D 多相孔隙图像(TiPM 2025)、有限岩心扫描+多模态融合的扩散 3D 重建(2025)。

## 精读卡（M5，阶段二 2D 扩散用 · 2026-06-25 读原文回填）

### 卡① Ho 2020 — DDPM（`literature/Ho2020_DDPM_diffusion.pdf`，NeurIPS2020）
- **机理**：*前向*固定加噪马尔可夫链 `q(xₜ|xₜ₋₁)=N(√(1-βₜ)xₜ₋₁, βₜI)`，闭式 `q(xₜ|x₀)=N(√ᾱₜ x₀,(1-ᾱₜ)I)`（αₜ=1-βₜ, ᾱₜ=∏αs，Eq2/4）；*反向*学高斯去噪 `pθ(xₜ₋₁|xₜ)=N(μθ,Σθ)`（Eq1）。
- **训练（Algorithm 1）**：采 x₀、t~U(1..T)、ε~N(0,I)，最小化 **ε-预测** 简化目标 `L_simple=E‖ε − εθ(√ᾱₜx₀+√(1-ᾱₜ)ε, t)‖²`（Eq14=去掉 Eq12 权重；消融 Table2 证 ε-预测+L_simple 样本质量最好）。
- **采样（Algorithm 2）**：xT~N(0,I)；`xₜ₋₁=1/√αₜ·(xₜ − (1-αₜ)/√(1-ᾱₜ)·εθ(xₜ,t)) + σₜz`。
- **关键超参**：T=1000；β 线性 1e-4→0.02；数据缩放到 **[-1,1]**；U-Net backbone（GroupNorm + 正弦时间嵌入 + 16×16 自注意力）。
- **对花页7**：阶段二 2D 孔隙结构生成器的训练目标/采样器直接照此；二值 pore/matrix 切片同样缩放到 [-1,1]。

### 卡② Mosser 2017 — GAN 重建多孔介质（`literature/Mosser2017_3D_porous_media_GAN.pdf`，PRE96:043309）
- **思路**：3D DCGAN 无监督学孔隙相空间分布，生成统计代表性体；附 GitHub。
- **评估指标集（= 本工作 eval 硬通货）**：
  - **孔隙度 φ**；
  - **两点概率/非中心协方差 `S₂(r)=P(x∈P, x+r∈P)`（Eq4）**：S₂(0)=φ、r→∞→φ²；沿 3 个笛卡尔方向 + 径向平均（捕捉各向异性）。
  - **Minkowski 泛函**（3D 共 4 个，算 3 个）：φ(0阶)、**比表面 SV**(1阶)、**Euler 特征 χV**(拓扑连通性)。
  - **单相方向渗透率**（格子 Boltzmann LBM）。
  - 验证对象：bead pack / Berea 砂岩 / Ketton 灰岩，生成体统计量与真实一致。
- **对花页7**：阶段二评估流水直接搬其指标集；2D MVP 取 2D 版（φ、S₂(r)、2D Euler χ=连通块−孔洞、比周长），渗透率(LBM)留待 3D/后期。
