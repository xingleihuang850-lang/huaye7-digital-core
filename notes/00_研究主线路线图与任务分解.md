# 研究主线路线图与任务分解

> **唯一主线 = 硕士论文《基于融合多尺度数据的页岩数字岩心-井筒建模》**。
> 贡献点是**生成式(扩散)+多模态融合+数字井筒**，**分割只是阶段一地基，不能一直停在多尺度分割**。
> codex 与我**协作互纠**（非竞赛）；共同目标=帮你完成花页7 多尺度研究 + 中期汇报。
> 一切**拆成小目标、逐个达成、都记录**，并挂**理论支撑**。

## 1. 开题 PPT 的四阶段（骨架，不可偏离）

| 阶段 | 内容 | 关键方法 |
|---|---|---|
| **一 分割** | SEM/QEMSCAN/CT/XRD → 矿物/孔隙分割 → 3D重建+孔隙网络 → 孔隙溶蚀/含水饱和度模型 → **电性模拟**(地层因素-孔隙度、电阻增大率-Sw) | 阈值 + nnU-Netv2；LBM/图形图像法 |
| **二 2D 生成** | 微米CT+FIB-SEM 配准/归一化 → 配对样本 → **DDPM 扩散** → 生成 2D 孔隙结构 | DDPM；视觉+参数(孔隙度/连通性)评估 |
| **三 多模态 3D 数字岩心** | 多尺度对齐(Elastix)+基础模型(CLIP/ViT/SAM)特征 → **条件扩散**(图像+测井曲线模态) → 3D重建 → 验证 | 条件扩散 + 多模态融合 |
| **四 数字井筒** | 高精度 3D 数字岩心 + 全直径岩心 → **多物理约束** → 数字井筒 | 多物理约束建模 |
