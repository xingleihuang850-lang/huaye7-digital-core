## 1. 文献依据

- **Ho2020 DDPM**（`literature/Ho2020_DDPM_diffusion.pdf`）：继续沿用前向加噪、ε-预测、反向采样的基本 DDPM 框架；B1 不改变扩散模型范式，只改变学习目标从二值 pore mask 转为连续灰度介质。
- **Mosser2017 3D porous media GAN**（`literature/Mosser2017_3D_porous_media_GAN.pdf`）：多孔介质生成的关键不是只看图，而是用 φ、S₂、Minkowski/Euler、连通性等物理统计量验收。B1 的思路与其一致：生成介质本体，再通过下游二值化/分割得到孔隙结构并评价。
- **Loucks2012 mudrock pore types**（`literature/Loucks2012_mudrock_pore_types.pdf`）：2.8μm micro-CT 只能覆盖 μm 级基质孔、较大粒内/粒间孔与微裂缝骨架；纳米级 OM 孔不是本阶段生成对象。B1 仍必须诚实表述为“μm 尺度灰度介质与孔隙网络”。
- **Kench2021 SliceGAN / Liu2025 Controlled Latent Diffusion / Sadeghkhani2025 PCP-GAN**：作为阶段三/条件生成/2D→3D 的后续依据；本步仍先做像素空间 2D 灰度 DDPM，不提前跳到 3D 潜扩散或孔隙度-深度条件生成。

## 2. 物理与地质依据

二值 pore mask 只告诉模型“哪里是孔”，但丢掉灰度介质中与孔隙聚集相关的上下文：

1. CT 灰度 `sus` 包含矿物/基质密度、孔隙边界、层理纹理和微裂缝形态的连续信号；这些信号决定孔隙更可能在哪里成簇，而不是独立撒点。
2. M7-v2 的 T* 标定能修 φ/S₂，却让 Euler 变差，说明“阈值”不是全部问题；孔隙空间组织本身缺少灰度语境。
3. M7-v3 200ep T=0 虽然 Euler 接近真实，但 φ 偏低，T* 拉回 φ 后 S₂/Euler 变差；这说明单纯学习二值标签的阈值/孔隙率偏置不稳定。
4. 花页7 4199.21m 页岩的阶段二对象是 2.8μm CT 的 μm 级孔隙/微裂缝网络；B1 生成灰度介质后再取孔隙，更符合“生成岩心介质 → 派生孔隙网络”的物理流程。


- 复用 M7 主链：tile=128、z_step=6、axes=z、min_valid=0.999、test_frac=0.2、seed=42。

- 若未来加入 y/x 切片增样，必须保持 test 只来自独立 z 留出块，不能让跨 z 的 y/x 切片污染测试区。

## 5. 灰度归一化策略

训练前必须先做只读分布检查，再定归一化参数；不要凭空假设灰度范围。

推荐第一版：

1. 在有效柱塞区域内统计 `sus` 灰度分布。
2. 用 robust percentile clip，例如 p1/p99 或 p0.5/p99.5；具体值由实际 histogram 决定。
3. clip 后线性映射到 `[-1,1]`。
4. meta 记录：
   - `gray_norm`: `percentile_clip_linear_to_minus1_1`
   - `p_low` / `p_high`
   - `raw_min` / `raw_max`
   - `valid_mask_rule`
   - `source_volume_path` / size / mtime / sha policy
   - `tile` / `z_step` / `axes` / `seed` / train-test counts

禁止：

- 不记录归一化参数；
- 用二值 pore 的 φ 去反推灰度阈值；
- 覆盖既有 `slices_ct28_128`；
- 把灰度 train/test 与 pore train/test 混名。

## 7. 下游取孔隙与评估口径

B1 采样得到灰度介质，不直接等于孔隙。必须分口径评估，禁止混读。

候选口径：

1. **B1-threshold**：对生成灰度做可解释阈值/标定，得到 pore。
2. **B1-nnUNet**：用 E3 可信 nnU-Netv2 同网格分割器对生成灰度取 pore；这一步是下游分析，不是生成模型本身。
3. **B1-reference-real**：真实 test 灰度经同一阈值/分割流程得到 pore，用于排除下游分割器本身的偏差。

每个口径单独目录：

```text
phase2/b1_gray_sus/eval_threshold/
phase2/b1_gray_sus/eval_nnunet/
phase2/b1_gray_sus/calib_threshold/
```

指标沿用 M7-v2/v3：

- φ mean/median/distribution
- S₂ radial rmse
- Euler χ mean/distribution
- max connected component fraction
- x/y penetrate ratio
- 与 50ep T=0、50ep T*、200ep T=0、200ep T*、real 同表比较

## 8. 通过/失败门槛

B1 不是只看图。第一版通过门槛：

- φ 不依赖极端阈值也能接近 real 6.4% 量级；
- S₂ rmse 不劣于 200ep T=0 的 0.00372，最好接近或优于 50ep T* 的 0.00242；
- Euler 分布接近 real 127.33，且不是靠 φ 偏低得到的伪接近；
- max CC frac 接近 real 0.0597，不再像 naive 那样接近 0，也不回到 50ep T=0 的过连通 0.1588；
- 样图灰度纹理与真实 ct28 test 灰度有可解释相似性。

失败处理：

- 若灰度纹理像噪声或孔隙不可分，先查归一化/数据切片，不急着改模型；
- 若灰度像真但 pore 提取不稳，优先比较 threshold 与 nnU-Net 下游口径；
- 若 B1 仍无法同时满足 φ/S₂/Euler/maxCC，再转 B2 `[sus,pore]` 双通道联合生成。
